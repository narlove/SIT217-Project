Coin Slot(eFSM) Machine

Built a coin slot machine with dynamic states based on state design pattern. The machine is represented as an object whereby it inherits properties through inheritance and then changes state through polymorphic implementation of inheritted classes.

Banking System

Designed an interface which uses dependecy injection, loose coupling of objects to replicate the functionalities of a banking system. Users can create accounts, indulge in transactions(withdraw, deposit and transfer). The system also has a rollback functionality allowing transactions to be reversed. The bank stores a history of all transactions.
